What term does Martin Fowler use to describe an agent harness?

---

Fowler defines a harness as a "cybernetic governor" that combines feed-forward and feedback to regulate agent behavior toward a desired state. The term comes from control theory: a cybernetic governor adjusts inputs based on outputs to maintain stable behavior. The other options describe related concepts (sandboxing, inference wrappers) but are not Fowler's definition.
