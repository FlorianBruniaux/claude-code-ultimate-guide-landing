What percentage of organizations have no formal agent identity strategy as of 2026, according to Strata Identity Research?

---

Strata Identity Research 2026 reports 77% of organizations have no formal agent identity strategy. Only 23% have a formal strategy, 44% are still using static API keys for agent authentication, and 18% rely on IAM trust inheritance with no per-agent scoping. This is the structural gap that motivates the Agent Identity Architect role. The Lethal Trifecta threat (private data + untrusted content + external communication) is well understood, but the defense has not been built at most organizations.
