How do you exit Plan Mode to start making changes?

---

Press Shift+Tab to cycle out of Plan Mode into an editing mode, or respond to Claude's prompt "Ready to implement this plan?" to transition. While in Plan Mode, Claude can only read and analyze - no modifications are allowed. There is no /execute slash command in Claude Code; exit is done through the Shift+Tab permission-mode cycle or by approving the plan. This staged approach ensures you understand the plan before execution.
